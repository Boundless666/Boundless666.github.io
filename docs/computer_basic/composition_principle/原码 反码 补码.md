计算机内部只有`加法`

​	3-2 = 3+(-2)

这就是为啥出现反码的原因  以及正数的反码是本身，而负数的反码除了符号位以外要取反的原因

| ---- | :-------: | :-------: |
| 原码 | 0000 0011 | 1000 0010 |
| 反码 | 0000 0011 | 1111 1101 |
